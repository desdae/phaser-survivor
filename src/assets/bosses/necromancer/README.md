# Necromancer Boss Asset Contract

This folder documents the imported necromancer boss art keys and the procedural keys that are generated in `GameScene.createTextures()`.

## Imported texture keys

- `boss-necromancer-idle`
- `boss-necromancer-idle-1`
- `boss-necromancer-idle-2`
- `boss-necromancer-cast`
- `boss-necromancer-summon`
- `boss-necromancer-pulse`
- `boss-necromancer-death`
- `boss-necromancer-portrait` is procedurally generated by `GameScene.createTextures()`

## Procedural generated keys

- `boss-necromancer-fallback-idle`
- `boss-necromancer-portrait`
- `boss-necro-aura`
- `boss-necro-summon-burst`
- `boss-necro-pulse-ring`

## Contract notes

- `boss-necromancer-fallback-idle` is a procedural fallback, not a required imported key.
- `boss-necromancer-portrait` is generated procedurally alongside the fallback art and should not be treated as imported-only.
- When imported art is missing, the scene falls back to the procedural keys generated in `GameScene.createTextures()`.
- The procedural fallback keys are generated at the following sizes: `72x92`, `92x120`, `96x96`, `96x96`, and `128x128`.
